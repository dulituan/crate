.. _ref-grant:

=========
``GRANT``
=========

Grants privilege to an existing user on a specific object.

Synopsis
========

.. code-block:: psql

  GRANT { { DQL | DML | DDL [,...] } | ALL [ PRIVILEGES ] }
  [{ ON SCHEMA schema_name |
    ON TABLE  table_ident }]
  TO user_name [, ...];

Description
===========

``GRANT`` is a management statement to grant one or many privileges
on a specific object to one or many existing users.

For usage of the ``GRANT`` statement see :ref:`administration-privileges`.

Parameters
==========

:schema_name: The name of an existing schema.
:table_ident: The ident of an existing table.
:user_name: The name of an existing user.
